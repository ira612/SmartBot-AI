# 🤖 SmartBot AI

> A modern Rule-Based AI Chatbot built with Python and Tkinter as part of the **DecodeLabs Generative AI Internship**.

---

## 📌 Project Overview

SmartBot AI is a desktop chatbot application developed using Python and Tkinter. It interacts with users through predefined rules using `if-else` logic and provides responses to greetings, AI-related questions, date/time queries, and more.

This project demonstrates GUI development, event handling, control flow, and user interaction in Python.

---

## ✨ Features

- 💬 Interactive chatbot interface
- 🎨 Modern Tkinter GUI
- ⌨️ Typing animation
- 📅 Date and time commands
- 🤖 AI and Python information
- 📖 Help menu
- 🗑️ Clear chat option
- ℹ️ About section

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/ira612/SmartBot-AI.git
```

### 2. Open the project folder

```bash
cd SmartBot-AI
```

### 3. Run the chatbot

```bash
python chatbot.py
```
